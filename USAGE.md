# AnswerFinder - Quick Usage Guide

## üöÄ Getting Started in 3 Steps

### Step 1: Install

1. Visit `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the AnswerFinder folder

### Step 2: Upload Data

1. Click the AnswerFinder icon
2. Click "Choose File"
3. Select your `.json` or `.txt` file
4. Wait for success message

### Step 3: Use It!

1. Highlight any question on a webpage
2. Right-click
3. Click "Search Answer"
4. View the answer instantly!

## üìù Creating Your Q&A File

### JSON Format (Recommended)

```json
[
  {
    "question": "What is the capital of France?",
    "answer": "Paris is the capital and largest city of France."
  },
  {
    "question": "Who invented the telephone?",
    "answer": "Alexander Graham Bell invented the telephone in 1876."
  }
]
```

### TXT Format (Simple)

```text
What is the capital of France?
Paris is the capital and largest city of France.

Who invented the telephone?
Alexander Graham Bell invented the telephone in 1876.
```

**Important:** Separate each Q&A pair with a blank line in TXT format!

## ‚öôÔ∏è Settings Explained

### Fuzzy Matching

- **What it does:** Handles typos and spelling variations
- **When to enable:** Almost always (recommended)
- **Example:** Matches "Wha is Paris?" with "What is Paris?"

### Partial Matching

- **What it does:** Finds answers even if you select only part of a question
- **When to enable:** When you frequently select partial text
- **Example:** "capital of France" matches "What is the capital of France?"

### Minimum Confidence

- **What it does:** Sets how strict the matching should be
- **Recommended:** 50% for balanced results
- **Lower (30-40%):** More results, less accurate
- **Higher (60-80%):** Fewer results, more accurate

### AI for Answers

- **What it does:** Uses AI when no local match is found
- **When to enable:** When you need answers to questions not in your database
- **Cost:** ~$0.0001 per query (nearly free!)
- **Limit:** 100 queries per day

## üí° Usage Tips

### For Best Results

1. **Format questions consistently** - Use similar phrasing in your Q&A file
2. **Include variations** - Add common ways people ask the same question
3. **Keep answers concise** - Short answers work best
4. **Use JSON for large datasets** - Better for 100+ questions
5. **Export regularly** - Backup your data using the Export button

### Keyboard Workflow

1. **Ctrl+F** (or **Cmd+F**) - Find text on page
2. **Triple-click** - Select entire question
3. **Right-click** - Open context menu
4. **Click "Search Answer"** - Get instant result

### AI Best Practices

- Enable AI only when needed to save on costs
- Review AI answers for accuracy
- Add good AI answers to your database (export, edit, re-upload)
- Set a custom proxy URL for better privacy

## üîß Troubleshooting

### "No match found" - What to do?

1. ‚úÖ Check if question exists in your database
2. ‚úÖ Lower minimum confidence setting
3. ‚úÖ Enable fuzzy and partial matching
4. ‚úÖ Enable AI for answers
5. ‚úÖ Try selecting more or less text

### Upload Failed - Common Fixes

1. ‚úÖ Check file format (must be .json or .txt)
2. ‚úÖ Validate JSON syntax (use jsonlint.com)
3. ‚úÖ Ensure file size is under 10MB
4. ‚úÖ Check for special characters or encoding issues

### AI Not Working - Solutions

1. ‚úÖ Verify "Enable AI for Answers" is checked
2. ‚úÖ Check proxy URL is correct
3. ‚úÖ Ensure you haven't exceeded daily limit (100 queries)
4. ‚úÖ Test proxy server is running (if using local server)
5. ‚úÖ Check browser console for errors (F12)

### Extension Not Loading

1. ‚úÖ Ensure Developer Mode is enabled
2. ‚úÖ Click "Reload" on the extension card
3. ‚úÖ Check for errors in chrome://extensions/
4. ‚úÖ Try removing and re-adding the extension

## üìä Understanding the Results

### Confidence Score

- **95-100%:** Exact or near-exact match
- **80-94%:** Strong keyword match
- **70-79%:** Fuzzy match with variations
- **60-69%:** Partial match
- **Below 60%:** Weak match (may not be shown)

### Match Type

- **Exact:** Perfect character match
- **Keyword:** Based on important words
- **Fuzzy:** Similar but not identical
- **Partial:** Matched part of the question
- **AI:** Generated by artificial intelligence

### Response Time

- **< 10ms:** Lightning fast local match
- **10-50ms:** Normal local search
- **2-4s:** AI-generated answer

## üîê Privacy & Security

### What Data is Stored?

- ‚úÖ Your Q&A pairs (locally in browser)
- ‚úÖ Settings and preferences (locally)
- ‚úÖ Search cache (locally, for speed)
- ‚ùå No data is sent to third parties (except AI queries)

### AI Privacy

- Questions sent to AI are routed through a proxy
- No personal information is transmitted
- Queries are not stored by the proxy
- Use your own proxy server for maximum privacy

### Data Control

- Export your data anytime
- Clear all data with one click
- No cloud sync or external backup
- Full control over your information
